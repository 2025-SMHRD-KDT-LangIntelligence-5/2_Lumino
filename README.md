# 📎 Averse (팀명: Lumino)
![Averse 플랫폼](https://github.com/user-attachments/assets/ad817e00-aabf-4c4e-959a-5f7444f6de16)




## 👀 서비스 소개
* 서비스 명 :  Averse (웹 기반 AI 솔루션 큐레이션 플랫폼)
* 서비스 설명 : AI 도구를 카테고리로 분류하여 사용자가 다양한 도구들을 직관적으로 탐색, 비교, 선택이 가능하며, 챗봇으로 AI 도구 추천을 받을 수 있고, 나만의 워크스페이스에 개발 환경을 저장 후 커뮤니티를 통한 추천 및 적용이 가능한 웹 기반 AI 도구 추천 플랫폼
<br>

## 📅 프로젝트 기간
2025.09.11 ~ 2025.11.26 (10주)
<br>

## ⭐ 주요 기능
* AI 도구 카테고리 분류 및 검색
* AI 도구 비교
* AI 도구 워크스페이스
* 커뮤니티 워크스페이스 추천
* RAG기반 챗봇
<br>
